# Community Theme Collection

Sindri's take on 15 community-favourite colour themes. Each theme ships as a matched pair — a colour theme and a matching file icon set — all in one click.

Install the collection from the Sindri Marketplace and apply whichever theme you like from **Settings → Appearance**.

## Icon assets

File and folder icons use **Tabler Icons** (MIT) with **Simple Icons** (CC0-1.0) for language and tool brand logos. See `sindri-community-icons-base/` for the shared icon template.

## Installing individual themes

Each theme is also installable on its own if you only want one without the full collection.
