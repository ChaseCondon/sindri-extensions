# Sindri Neon Icons

A vibrant neon file icon theme for Sindri IDE.

Language icons use bold label glyphs with saturated language-brand colours on dark backgrounds — easy to scan at a glance in the file tree.

**Covers:** JS · TS · JSX · TSX · Python · Rust · Go · JSON · TOML · YAML · Markdown · CSS · SCSS · HTML · SVG · Shell · lock · .env · git
