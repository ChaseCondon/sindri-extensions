# Sindri Aurora

A deep purple and teal dark theme for Sindri IDE.

- **Accent:** Violet `#B06CFF`
- **Strings / literals:** Teal `#00D4B4`
- **Keywords:** Soft lavender `#C490FF`
- **Background:** Deep ink `#0B0A18`

Designed for long sessions — high contrast where you need it, muted chrome around it.
