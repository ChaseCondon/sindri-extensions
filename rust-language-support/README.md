# Rust Language Support

Full Rust development support for Sindri IDE.

- **Syntax** — Tree-sitter grammar (`tree-sitter-rust`)
- **LSP** — rust-analyzer (hover, completions, diagnostics, go-to-definition)
- **DAP** — LLDB debugger integration

**Status:** Scaffolded — requires the Sindri extension host (in development). The manifest documents the intended contribution shape; the `dist/extension.js` entry point and grammar WASM file will be added once the host is available.
