# Aurora Pack

The complete Aurora experience for Sindri IDE — install all three Aurora extensions in one click.

## What's included

| Extension | Type | Description |
| --- | --- | --- |
| Sindri Aurora | Colour Theme | Deep purple and teal dark theme |
| Aurora File Icons | File Icon Theme | File tree icons in the Aurora palette |
| Aurora UI Icons | UI Icon Theme | Activity bar icons tuned for the Aurora accent |

## Installation

Click **Install** to install all three extensions together. Switch to Aurora Dark under Settings → Appearance → Colour theme, then select Aurora File Icons and Aurora UI Icons in the icon dropdowns.

## About extension packs

Extension packs are a way to bundle related extensions that are better together. Installing a pack installs each member extension individually — you can manage, update, or uninstall them independently afterward.
