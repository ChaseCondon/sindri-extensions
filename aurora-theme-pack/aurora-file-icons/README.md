# Aurora File Icons

A file icon set designed to complement the Sindri Aurora colour theme. Violet, lavender, and teal on deep ink backgrounds — seamless with the Aurora palette.

## Covered file types

| Extensions | Colour |
| --- | --- |
| `.js` `.mjs` | Lavender `#C490FF` |
| `.ts` `.mts` | Violet `#B06CFF` |
| `.jsx` `.tsx` | Teal `#00D4B4` |
| `.rs` | Light purple `#E0B0FF` |
| `.py` | Teal `#00D4B4` |
| `.go` | Sky blue `#79B8FF` |
| `.json` | Violet `#B06CFF` |
| `.md` | Lavender `#C490FF` |
| `.git*` | Coral `#FF5A6E` |

## Part of the Aurora Pack

This icon set is bundled together with the Aurora colour theme and Aurora UI Icons in the **Aurora Pack** extension pack.

Install all three together by searching for "Aurora Pack" in the marketplace.
